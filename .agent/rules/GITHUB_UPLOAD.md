# Инструкция по загрузке на GitHub

## Быстрые команды

### 1. Обновить версию плагина

Перед загрузкой изменений обновите версию плагина в файле `avito-uralgips.php`:

1. **Строка 6** — заголовок плагина:
   ```php
   * Version: 1.0.1
   ```

2. **Строка 25** — константа версии:
   ```php
   define('WC_AVITO_VDOM_VERSION', '1.0.1');
   ```

> ⚠️ **Важно**: Обе версии должны совпадать!

---

### 2. Загрузить изменения
```bash
git add .
git commit -m "Описание изменений"
git push
```

### 3. Проверить статус
```bash
git status
```

### 4. Посмотреть историю
```bash
git log --oneline -10
```
